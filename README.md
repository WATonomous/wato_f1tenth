# Jetson Production Branch

this is the jetson Production branch. for all future developmnet and testing on the jetson, please use this brach as the starting point and create a brach out of. 
this branch will always act as a template for starting developmnet of new features. 

## what basic functions does this brach provide ? 

1. offers the diana bash script docker utlity (for more into do ```./diana --h or ./diana --help```)
2. support on how to connect to the jetson remotely with GUI 
3. full working state estimation stack (odom -> ekf -> pf)
4. teleop support with gamepads and keyboards
5. host of detailed tutorials in the tutorials directory 


## Final Remarks 
all the tutorials were made using linux, thus I can't garuntee all of the stuff covered will work on windows. 
however most of the remote stuff with nomachine should work fine with windows.
